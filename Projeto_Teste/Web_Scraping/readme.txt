## Passo a Passo para Rodar o Script

1. **Instalar as Bibliotecas Necessárias**\
   Abra o terminal ou prompt de comando e execute:

   ```bash
   pip install requests beautifulsoup4
   ```

2. **Executar o Script**\
   Certifique-se de que o Python está instalado e execute o script com:

   ```bash
   python script.py
   ```

3. **Resultados**

   - O script baixará os anexos I e II da página da ANS.
   - Os arquivos serão salvos na pasta `anexos_ans`.
   - Um arquivo ZIP chamado `anexos_ans.zip` será gerado com os PDFs baixados.
